# 🚀 SolveSphere

**SolveSphere** is an AI-powered collaborative problem-solving platform where users can explore real-world challenges, build innovative solutions, and get intelligent assistance using AI.

---

## 🌍 Overview

SolveSphere bridges the gap between **problem identification and solution building** by combining:

* 🌐 Real-world problems (Local + Global)
* 🤖 AI-powered assistance (Gemini integration)
* 🧠 Technical & non-technical solution building
* 📊 Smart evaluation & scoring system
* 🎯 Interactive quizzes (like LeetCode-style)

---

## ✨ Key Features

### 🧩 Problem Explorer

* Browse 150+ problems
* Categories: **Local (e.g., traffic, pollution)** & **Global**
* Difficulty levels: Easy, Medium, Hard

---

### 🛠️ Solution Builder

* Create structured solutions
* Choose between:

  * 📄 Documentation
  * 🎨 Visual Blueprint

---

### 🤖 AI Assistant (Gemini Powered)

* Ask any question (general or problem-specific)
* Get intelligent suggestions
* Improve solutions dynamically

---

### 🧪 Quiz Module

* 🧑‍💻 Technical (LeetCode-style)
* 📊 Non-Technical (domain-based: accounting, agriculture, etc.)
* Difficulty-based quizzes

---

### 🏆 Evaluation & Scoring

* Strict scoring system
* Based on relevance, structure, and feasibility
* Enables:

  * Badges 🏅
  * Rankings 📈
  * Leaderboards

---

### 👤 User Features

* Authentication system
* Portfolio of submitted solutions
* Progress tracking

---

## 🧠 Tech Stack

### Frontend

* React (Vite)
* TypeScript
* Tailwind CSS

### Backend

* Supabase (Database + Auth + Edge Functions)
* Gemini API (AI integration)

---

## 🔐 Environment Setup

Create a `.env` file:

```env
VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
VITE_GEMINI_API_KEY=your_key
```

---

## 🚀 Run Locally

```bash
npm install
npm run dev
```
